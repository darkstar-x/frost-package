## Getting Started

#### Prerequisites for frost pkg
`
bash >= 4.4.18
`
`
p7zip >= 16.02
`
#### Manual installation

Clone the repository
```sh
$ git clone https://github.com/hxvrmxnv/frost_pkg.git
```
As ROOT, create the directories
```sh
# mv frost /usr/sbin
```
